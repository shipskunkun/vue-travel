## 1.组件间传值 

直接从组件props 的数据，是会随着 props的改变而改变的

但是，如果你想在子组件中对 props 数据进行处理，是会提示错误的

此时需要，this.count = number , 进行转换

但是这样，props number 变化，子组件的 count 就不会变化了

需要对 number 进行watch

```
popupType(val) {
	this.count = val;
}
```
